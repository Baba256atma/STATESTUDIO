# Nexora Debug Workflow

## Debug Flow
1. Identify symptom
2. Trace logs
3. Locate source layer (Chat / Panel / Scene / Backend)
4. Apply minimal fix

## Required Debug Traces
- `[Nexora][PanelFlow]`
- `[Nexora][Reaction]`
- `[Nexora][Router]`
- `[Nexora][Scene]`

## Common Issues
- panel flash / disappear
- scene overwrite
- empty panel rendering
- routing mismatch
